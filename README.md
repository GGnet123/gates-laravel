<h2>Тестовое задание (криптошлюз):</h2>

Проекты желательно поднять на **docker**.

Для старта проекта просто выполните команду: <br>
``make setup``

Она сама поднимет докер и создаст `.env` файл из `.env.example`

Планировщик сервис сразу же начнет запускать очереди 
на индексирование и скан блоков.

Для ручного добавления индексации в очередь выполните 

`make index` или же `docker compose exec app php artisan blockchain:index --base_gate=eth_sepolia`

Для тригера запроса на создание адресса в го проект
выполните `make create-address`

Для получения горячего кошелька для выводов
выполните `make hot-address`
